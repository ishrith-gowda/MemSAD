\title{\memsad{}: Gradient-Coupled Anomaly Detection for Memory Poisoning in Retrieval-Augmented Agents}

\begin{abstract}
Persistent external memory enables LLM agents to maintain context across sessions, yet its security properties remain formally uncharacterized.
We formalize memory poisoning attacks on retrieval-augmented agents as a Stackelberg game and present a unified evaluation framework spanning three attack classes with escalating access assumptions. Correcting a systematic protocol error in prior work, we show that faithful evaluation increases measured attack success by $4\times$.
Our primary contribution is \memsad{} (\emph{Semantic Anomaly Detection}), a calibration-based defense grounded in a \emph{gradient coupling theorem}: under encoder regularity, the anomaly score gradient and the retrieval objective gradient are provably identical, so any continuous perturbation that reduces detection risk necessarily degrades retrieval rank. This coupling yields a certified detection radius that guarantees correct classification regardless of adversary strategy.
We prove minimax optimality via Le Cam's method, showing that any threshold detector requires $\Omega(1/\rho^2)$ calibration samples and \memsad{} achieves this up to $\log(1/\delta)$ factors. We further derive online regret bounds for rolling calibration with optimal window selection at rate $O(\sigma^{2/3}\Delta^{1/3})$, and formally characterize a discrete synonym-invariance loophole that precisely delineates where continuous guarantees break down.
Experiments on a $3 \times 5$ attack-defense matrix with bootstrap confidence intervals, Bonferroni-corrected hypothesis tests, and Clopper-Pearson validation (20 trials, $n=1{,}000$) confirm the theory: composite defenses achieve $\tpr = 1.00$, $\fpr = 0.00$ across all attacks, while synonym substitution evades detection at $\Delta\asrr \approx 0$, delineating an open frontier for memory security.
\end{abstract}

\section{Introduction}
\label{sec:intro}

Modern LLM agents persist context across sessions through external memory systems such as Mem0~\citep{mem0}, A-MEM~\citep{amem}, and MemGPT~\citep{memgpt2023}, storing user preferences and factual knowledge as dense vector embeddings.
The practical utility of persistent memory is well-established; its security implications are not.
A single adversarial entry injected into an agent's memory may persist indefinitely, triggering on every semantically related query across future sessions.
Unlike prompt injection, which requires per-interaction access, memory poisoning \emph{persists and scales}: the adversary acts once, and the attack surface compounds with each new query.

Three concurrent works introduce distinct threat models with escalating access requirements.
\agentpoison{}~\citep{chen2024agentpoison} formulates trigger optimization as a constrained gradient problem over DPR~\citep{karpukhin2020dense} embeddings ($\alpha = \textsc{write}$).
\minja{}~\citep{dong2025minja} extends the setting to query-only access via progressive-shortening indication ($\alpha = \textsc{query}$).
\injecmem{}~\citep{injecmem2026} achieves single-interaction injection via retriever-agnostic anchors ($\alpha = \textsc{single}$).
Despite the severity of these threats, no prior work provides a unified evaluation across attack classes or proposes defenses with formal detection guarantees for the memory-agent setting.
Existing RAG defenses~\citep{zou2024poisonedrag, chaudhari2024phantom, xiang2024robustrag} assume corpus-level access and offline cleaning, neither of which applies to the streaming ingestion model of agent memory.

\paragraph{Contributions.}
Our five contributions span theory, algorithm design, and empirical methodology.
\textbf{(1)}~\emph{Formal threat model.} Stackelberg game formulation (Definitions~\ref{def:threat_model}--\ref{def:stackelberg}) with equilibrium existence (Proposition~\ref{prop:equilibrium}), and a protocol distinction showing that evaluating \agentpoison{} under its intended triggered-query protocol raises $\asrr$ from $0.25$ to $1.00$.
\textbf{(2)}~\emph{\memsad{} defense.} A calibration-based detector (Definition~\ref{def:sad}) with the gradient coupling theorem (Theorem~\ref{thm:gradient_coupling}) proving continuous evasion impossible, and a certified detection radius (Lemma~\ref{lem:certified}) providing checkable, per-entry guarantees.
\textbf{(3)}~\emph{Theoretical foundations.} Information-theoretic lower bound via Le Cam's method (Theorem~\ref{thm:minimax_lower}) proving minimax optimality up to a $\log(1/\delta)$ factor; finite-sample calibration bounds (Theorem~\ref{thm:calibration_bound}); online regret bounds (Theorem~\ref{thm:regret}) with optimal window at rate $O(\sigma^{2/3}\Delta^{1/3})$; and a Wasserstein generalization bound (Theorem~\ref{thm:generalization}) for transfer to unseen attacks.
\textbf{(4)}~\emph{Impossibility characterization.} Formal analysis of the discrete synonym-invariance loophole (Proposition~\ref{prop:synonym}) with tight perturbation bounds, characterizing exactly where continuous guarantees break down.
\textbf{(5)}~\emph{Rigorous evaluation.} A $3 \times 5$ attack-defense matrix with Bonferroni-corrected hypothesis testing ($\alpha' = 0.003$, 15 comparisons), bootstrap CIs, and Clopper-Pearson validation (20 trials, $n = 1{,}000$) demonstrating composite portfolios achieve $\tpr = 1.00$, $\fpr = 0.00$ across all attacks.

\section{Related work}
\label{sec:related}

\paragraph{Memory poisoning attacks.}
\citet{chen2024agentpoison} report $\asrr \geq 0.80$ via trigger optimization over DPR~\citep{karpukhin2020dense} embeddings under the triggered-query protocol, requiring write access to the memory store.
\citet{dong2025minja} exploit auto-storage via progressive-shortening indication ($\isr = 98.2\%$) under query-only access.
\citet{injecmem2026} achieve single-interaction injection via retriever-agnostic anchors, the weakest attacker assumption.
\citet{memorygraft2025} demonstrate experience-grafting attacks exploiting union retrieval.
Concurrent work includes TrojanRAG~\citep{trojanrag2024} (joint backdoor), CorruptRAG~\citep{corruptrag2025} (single-text poisoning), BadRAG~\citep{xue2024badrag} (retrieval vulnerability analysis), Phantom~\citep{chaudhari2024phantom} (general trigger attacks), and jamming attacks~\citep{shafran2025blocker} (denial-of-service). Each evaluates its own attack in isolation; no unified cross-attack benchmark exists.
Unlike training-time backdoor attacks that embed triggers via parameter poisoning~\citep{wu2024badagent}, memory poisoning targets the retrieval index directly at inference time, requiring no access to model weights and persisting across sessions.
On the defense side, A-MemGuard~\citep{amemguard2025} proposes proactive defenses via query rewriting, RevPRAG~\citep{tan2025revprag} detects poisoning through LLM activation analysis, ReliabilityRAG~\citep{reliabilityrag2025} provides provable robustness via graph-theoretic MIS, and SeCon-RAG~\citep{seconrag2025} uses entity-intent extraction. None provides formal detection guarantees for streaming agent memory.

\paragraph{RAG corpus poisoning and certified defenses.}
\citet{zou2024poisonedrag} optimize adversarial documents via HotFlip~\citep{ebrahimi2018hotflip} ($> 90\%$ ASR on million-document corpora).
\citet{xiang2024robustrag} provide certified robustness via \emph{isolate-then-aggregate}, but require post-retrieval processing of $k$ passages.
RAGDefender~\citep{ragdefender2025} applies post-retrieval passage scoring; SPECTRE~\citep{hayase2021spectre} uses spectral detection requiring full corpus access ($O(|\cM|^2 d)$).
Our work differs along three axes: (i)~defenses operate at \emph{write time} on individual entries ($O(md)$ per entry), (ii)~we require no corpus-level access, and (iii)~we provide both a certified detection radius (Lemma~\ref{lem:certified}) and minimax optimality (Theorem~\ref{thm:minimax_lower}), whereas prior certificates apply post-retrieval.

\paragraph{Game-theoretic security and adaptive evaluation.}
\citet{tramer2020adaptive} formalize best practices for adaptive attack evaluation; \citet{carlini2017evaluating} demonstrate that defenses without formal guarantees are routinely broken.
Our Stackelberg formulation (Definition~\ref{def:stackelberg}) instantiates both principles: the adversary optimizes with full knowledge of $(\hat{\mu}, \hat{\sigma}, \kappa)$, and the gradient coupling theorem (Theorem~\ref{thm:gradient_coupling}) provides the formal guarantee that \citet{carlini2017evaluating} require, while Proposition~\ref{prop:synonym} honestly characterizes where it fails.

\section{Formal threat model}
\label{sec:threat}

\begin{definition}[Memory-augmented agent]
\label{def:agent}
A \emph{memory-augmented agent} is a tuple $(\cA, \cM, \enc, k)$ where $\cA$ is an LLM, $\cM \subset \cV^*$ is a persistent memory store over vocabulary $\cV$, $\enc : \cV^* \to \RR^d$ is an embedding function, and $k \in \NN$ is the retrieval depth. At each interaction with query $q$, the agent retrieves
\begin{equation}
  \Ret(\cM, q, k) = \argmax_{\substack{S \subseteq \cM \\ |S| = k}} \sum_{m \in S} \cosim(\enc(q), \enc(m)),
  \label{eq:retrieval}
\end{equation}
where $\cosim(u, v) \coloneqq u^\top v / (\|u\| \|v\|)$ denotes cosine similarity.
\end{definition}

\begin{definition}[Memory poisoning threat model]
\label{def:threat_model}
An \emph{adversary} $\cS = (\cQ_v, n, \alpha)$ targets victim queries $\cQ_v \subset \cV^*$ by injecting $\cP = \{p_1, \ldots, p_n\}$ into $\cM$ under access model $\alpha \in \{\textsc{write}, \textsc{query}, \textsc{single}\}$ to maximize:
\begin{align}
  \asrr(\cP, \cQ_v) &\coloneqq |\{q \in \cQ_v : \cP \cap \Ret(\cM \cup \cP, q, k) \neq \emptyset\}| / |\cQ_v|, \label{eq:asrr} \\
  \asra(\cP) &\coloneqq \PP[\cA \text{ executes adversarial action} \mid \text{poison retrieved}], \label{eq:asra} \\
  \asrt(\cP, \cQ_v) &\coloneqq \asrr \cdot \asra. \label{eq:asrt}
\end{align}
\end{definition}

\begin{definition}[Stackelberg game formulation]
\label{def:stackelberg}
The memory poisoning interaction is a Stackelberg game $\cG = (\cS, \cD, u_{\cS}, u_{\cD})$ where the defender (leader) commits to a detection policy $\pi_{\cD} : \cV^* \to \{0, 1\}$ and the adversary (follower) best-responds:
\begin{align}
  \pi_{\cD}^* &= \argmin_{\pi_{\cD} \in \Pi} \; \max_{\cP \in \cB(\pi_{\cD})} \; \asrr(\cP, \cQ_v), \label{eq:leader} \\
  \cB(\pi_{\cD}) &= \argmax_{\cP \subseteq \cV^*,\, |\cP| \leq n} \; \asrr(\cP \setminus \{p : \pi_{\cD}(p) = 1\}, \cQ_v), \label{eq:follower}
\end{align}
where $\Pi = \{\pi(c) = \mathbf{1}[s(c; \cH) > \tau] : \tau > 0\}$ is the class of threshold detectors.
The defender's utility is $u_{\cD} = -\asrr - \lambda \cdot \fpr$ ($\lambda > 0$) and the adversary's is $u_{\cS} = \asrt$.
\end{definition}

\begin{assumption}[Encoder regularity]
\label{ass:encoder}
$\enc : \cV^* \to \RR^d$ satisfies: (i) $\|\enc(x)\| > 0$ for all $x$; (ii) differentiability w.r.t.\ the output embedding; (iii) L2-normalization: $\|\enc(x)\| = 1$ (w.l.o.g.).
\end{assumption}

Under Assumption~\ref{ass:encoder}(iii), $\cosim(\enc(q), \enc(m)) = \enc(q)^\top \enc(m)$ and retrieval reduces to maximum inner product search.

\section{\memsad{}: Semantic Anomaly Detection}
\label{sec:sad}

\subsection{Algorithm and formal definition}

The key insight is that memory poisoning succeeds \emph{because} adversarial entries are semantically close to victim queries; this closeness is itself a detectable signal.

\begin{definition}[\memsad{} detector]
\label{def:sad}
Let $\cH = \{q_1, \ldots, q_m\}$ be a rolling query history (FIFO, capacity $m_{\max}$) and $\cM_{\text{ref}} \subseteq \cM$ a benign reference corpus. The anomaly score is $s(c; \cH) \coloneqq \max_{q \in \cH} \cosim(\enc(c), \enc(q))$, calibrated via $\hat{\mu}, \hat{\sigma}$ of $\{s(m; \cH) : m \in \cM_{\text{ref}}\}$. Entry $c$ is flagged if $s(c; \cH) > \hat{\mu} + \kappa \hat{\sigma}$ for threshold parameter $\kappa > 0$.
The \emph{combined scoring mode} averages max-query and mean-query similarity, capturing both targeted attacks (high max similarity) and distributed attacks (elevated mean similarity):
\begin{equation}
  s_{\text{comb}}(c; \cH)
  \coloneqq \tfrac{1}{2} \max_{q \in \cH} \cosim(\enc(c), \enc(q))
  + \tfrac{1}{2} \cdot \frac{1}{|\cH|}\sum_{q \in \cH} \cosim(\enc(c), \enc(q)).
  \label{eq:sad_combined}
\end{equation}
\end{definition}

\begin{algorithm}[t]
\caption{\memsad{} pre-ingestion filtering}
\label{alg:sad}
\begin{algorithmic}[1]
\REQUIRE candidate $c$, query history $\cH$, parameters $(\hat{\mu}, \hat{\sigma})$, threshold $\kappa$
\STATE $s \leftarrow s_{\text{comb}}(c; \cH)$ \hfill $\triangleright$ $O(md)$ time
\IF{$s > \hat{\mu} + \kappa \cdot \hat{\sigma}$}
  \STATE \textbf{reject} $c$
\ELSE
  \STATE \textbf{accept} $c$ into $\cM$
\ENDIF
\end{algorithmic}
\end{algorithm}

\subsection{Gradient coupling theorem}
\label{sec:gradient}

Can the defender design a detection objective that decouples from retrieval, allowing the adversary to optimize retrieval without increasing detectability? The following theorem proves impossibility: any detection function monotone in cosine similarity inherits the retrieval gradient, and conversely, any detection function that decouples from retrieval necessarily misses high-similarity adversarial entries.

\begin{theorem}[Gradient coupling]
\label{thm:gradient_coupling}
Let $\enc$ satisfy Assumption~\ref{ass:encoder}, let $e_c \coloneqq \enc(c) \in \RR^d$, and $q^* \coloneqq \argmax_{q \in \cH} \cosim(e_c, \enc(q))$. Define the retrieval objective $\cR(e_c) \coloneqq \cosim(e_c, \enc(q^*))$ and let $\cD : \RR^d \to \RR$ be any differentiable detection function satisfying $\cD = g \circ \cR$ for differentiable, monotone increasing $g$ with $g' > 0$. Then:
\begin{equation}
  \nabla_{e_c} \cD(e_c)
  = g'(\cR(e_c)) \cdot \nabla_{e_c} \cR(e_c)
  = g'(\cR(e_c)) \cdot \frac{\enc(q^*) - (e_c^\top \enc(q^*)) e_c}{\|e_c\|^2 \|\enc(q^*)\|}.
  \label{eq:gradient}
\end{equation}
Since $g' > 0$, for any perturbation $\delta \in \RR^d$: $\langle \delta, \nabla \cD \rangle < 0 \iff \langle \delta, \nabla \cR \rangle < 0$.
Conversely, if $\cD$ is not monotone in $\cR$, there exist adversarial embeddings $e_c$ with $\cR(e_c) > \tau_{\text{ret}}$ yet $\cD(e_c) < \tau_{\text{det}}$, i.e., successful retrieval without detection.
\end{theorem}

\begin{proof}
The forward direction follows from the chain rule: $\nabla \cD = g'(\cR) \cdot \nabla \cR$ with $g' > 0$ preserving the sign of every directional derivative. For the converse, non-monotonicity of $\cD$ in $\cR$ implies the existence of $e_c, e_c'$ with $\cR(e_c') > \cR(e_c)$ but $\cD(e_c') < \cD(e_c)$; by continuity, a level set of $\cD$ at $\tau_{\text{det}}$ intersects the region $\{\cR > \tau_{\text{ret}}\}$ for appropriate thresholds.
\end{proof}

\noindent \memsad{}'s max-cosine scoring (Definition~\ref{def:sad}) instantiates $g = \mathrm{id}$, achieving maximal coupling. In particular, $\cD(e_c') < \cD(e_c)$ (evasion) implies $\cR(e_c') < \cR(e_c)$ (retrieval degradation) to first order; the second-order tradeoff surface is analyzed in Appendix~\ref{app:tradeoff_surface}.

\subsection{Certified detection guarantee}
\label{sec:certified}

We now provide an \emph{instance-specific} guarantee: when the adversarial-benign similarity gap exceeds the calibration uncertainty, \memsad{} certifiably detects the adversarial passage.

\begin{lemma}[Certified detection radius]
\label{lem:certified}
Let $\enc$ satisfy Assumption~\ref{ass:encoder} and let $c$ be an adversarial passage achieving $\rank(\enc(c), \cM) \leq k$ for victim query $q^*$. Define the \emph{benign similarity ceiling} $\bar{s} \coloneqq \max_{m \in \cM_{\text{ref}}} \cosim(\enc(m), \enc(q^*))$, the adversarial similarity $s_{\text{adv}} \coloneqq \cosim(\enc(c), \enc(q^*))$, and the \emph{gap} $\Delta_s \coloneqq s_{\text{adv}} - \bar{s}$.
If the calibration satisfies $|\tau_N - \tau^*| \leq \eta$ (Theorem~\ref{thm:calibration_bound}) and the gap exceeds
\begin{equation}
  \Delta_s > \kappa\hat{\sigma} + \eta,
  \label{eq:certified_radius}
\end{equation}
then \memsad{} detects $c$ with certainty: $s(c; \cH) > \hat{\mu} + \kappa\hat{\sigma}$.
\end{lemma}

\begin{proof}
Since $c$ achieves $\rank \leq k$, we have $s(c; \cH) \geq s_{\text{adv}} = \bar{s} + \Delta_s$. By the gap assumption, $\bar{s} \geq \tau^* = \mu + \kappa\sigma$ (the benign ceiling exceeds the population threshold, which holds whenever the reference corpus contains a near-query entry, as is typical for $|\cM_{\text{ref}}| \gg 1$). Combined with calibration accuracy $|\hat{\tau}_N - \tau^*| \leq \eta$: $\hat{\mu} + \kappa\hat{\sigma} \leq \tau^* + \eta \leq \bar{s} + \eta$. Thus $s(c; \cH) \geq \bar{s} + \Delta_s > \bar{s} + \kappa\hat{\sigma} + \eta \geq \hat{\mu} + \kappa\hat{\sigma}$.
\end{proof}

Figure~\ref{fig:certified_radius} illustrates the certified radius.
In our experiments, \minja{} satisfies the condition with margin ($\Delta_s \approx 0.15 > \kappa\hat{\sigma} + \eta \approx 0.08$); \agentpoison{} under triggered calibration achieves $\Delta_s \approx 0.12 > 0.06$.

\subsection{Information-theoretic detection lower bound}
\label{sec:lower_bound}

\begin{theorem}[Minimax detection lower bound]
\label{thm:minimax_lower}
Let $\cF$ be the class of threshold detectors $\pi(c) = \mathbf{1}[s(c; \cH) > \tau]$. Let $P_0 = \cN(\mu_0, \sigma^2)$, $P_1 = \cN(\mu_1, \sigma^2)$ be Gaussian benign and poisoned score distributions with common variance $\sigma^2$, and $\rho \coloneqq |\mu_1 - \mu_0| / \sigma$ the signal-to-noise ratio. (The Gaussian assumption is validated empirically in Appendix~\ref{app:proof_minimax}.) For any detector $\pi \in \cF$ calibrated on $N$ i.i.d.\ samples from $P_0$:
\begin{equation}
  \inf_{\pi \in \cF} \left[\fpr(\pi) + (1 - \tpr(\pi))\right]
  \geq 1 - \frac{\rho\sqrt{N}}{2}.
  \label{eq:minimax_lower}
\end{equation}
Reliable detection ($\fpr + (1 - \tpr) \leq \epsilon$) requires $N \geq 4(1 - \epsilon)^2 / \rho^2$.
\end{theorem}

\begin{proof}[Proof sketch]
Apply Le Cam's two-point method~\citep{lecam1973convergence,yu1997assouad}. Pinsker gives $\TV(P_0^N, P_1^N) \leq \rho\sqrt{N}/2$; Le Cam yields Eq.~\eqref{eq:minimax_lower}. Full proof in Appendix~\ref{app:proof_minimax}.
\end{proof}

\noindent \memsad{} achieves reliable detection with $N = O(\log(1/\delta)/\rho^2)$ samples (Theorem~\ref{thm:calibration_bound}), matching the $\Omega(1/\rho^2)$ lower bound up to a $\log(1/\delta)$ factor, establishing minimax optimality.

\subsection{Calibration bounds and online regret}

\begin{theorem}[Calibration bound]
\label{thm:calibration_bound}
For $N$ i.i.d.\ benign scores with threshold $\tau_N = \hat{\mu}_N + \kappa\hat{\sigma}_N$, with probability $\geq 1 - \delta$:
$|\tau_N - \tau^*| \leq \sigma \bigl(\frac{1}{\sqrt{N}} + \frac{\kappa}{\sqrt{2(N-1)}}\bigr) \sqrt{2 \log(4/\delta)}$,
where $\tau^* = \mu + \kappa\sigma$. Proof in Appendix~\ref{app:proof_calibration}.
\end{theorem}

\begin{theorem}[Online calibration regret]
\label{thm:regret}
Suppose queries arrive sequentially with per-step drift $\|\EE[\enc(q_t)] - \EE[\enc(q_{t+1})]\| \leq \Delta$. The cumulative regret of \memsad{}'s rolling window satisfies:
\begin{equation}
  \Regret(T) = \sum_{t=1}^{T} |\hat{\tau}_t - \tau_t^*|
  \leq T \cdot O\!\left(\frac{\sigma}{\sqrt{m_{\max}}} + m_{\max} \cdot \Delta \right).
  \label{eq:regret}
\end{equation}
The optimal window $m_{\max}^* = \Theta((\sigma/\Delta)^{2/3})$ minimizes per-step regret to $O(\sigma^{2/3}\Delta^{1/3})$. Non-asymptotic convergence bounds are in Appendix~\ref{app:proof_regret}.
\end{theorem}

\subsection{Generalization to unseen attacks}

\begin{theorem}[Generalization bound]
\label{thm:generalization}
Let $P_{\text{new}}$ be the score distribution of a novel attack with $W_1(P_{\text{new}}, P_j) \leq \epsilon_W$ for some observed attack $P_j$. Then $|\tpr_{\text{new}} - \tpr_j| \leq \epsilon_W / \hat{\sigma} + O(1/\sqrt{N})$, where $N$ is the calibration size. Proof via Kantorovich-Rubinstein duality in Appendix~\ref{app:proof_generalization}.
\end{theorem}

\noindent In our experiments, \minja{} and \injecmem{} have $W_1 \approx 0.03\hat{\sigma}$, predicting $|\Delta\tpr| \leq 0.03$. The observed $\Delta\tpr = 0.00$ is well within this bound.

\subsection{The discrete synonym loophole}

\begin{proposition}[Synonym-invariance loophole]
\label{prop:synonym}
Let $\enc$ satisfy $\|\enc(\mathrm{sub}(c, w, w')) - \enc(c)\| \leq \epsilon_{\text{syn}}$ for all synonym pairs $(w, w')$. For $r$ substitutions yielding $c'$:
\begin{equation}
  |\cR(c') - \cR(c)| \leq r \epsilon_{\text{syn}}, \qquad
  |s(c'; \cH) - s(c; \cH)| \leq r \epsilon_{\text{syn}}.
  \label{eq:synonym_bounds}
\end{equation}
When $\epsilon_{\text{syn}} \approx 0$, the adversary evades detection while preserving retrieval rank through discrete lexical substitution that bypasses gradient coupling.
\end{proposition}

\begin{proof}
Cauchy-Schwarz gives $|\enc(q)^\top(\enc(c') - \enc(c))| \leq \|\enc(c') - \enc(c)\|$, and triangle inequality over $r$ substitutions yields $\|\enc(c') - \enc(c)\| \leq r\epsilon_{\text{syn}}$.
\end{proof}

This does \emph{not} contradict Theorem~\ref{thm:gradient_coupling}: the coupling holds for \emph{continuous} perturbations in $\RR^d$, while synonym substitution operates in the \emph{discrete} token space where $\epsilon_{\text{syn}} \approx 0$ for paraphrase-trained encoders. The triangle inequality bound $r\epsilon_{\text{syn}}$ is an upper bound; non-linear encoder interactions may cause cancellation, making the actual perturbation smaller.

\section{Additional defenses}
\label{sec:other_defenses}

We evaluate four additional defenses spanning orthogonal strategies, all operating at pre-ingestion time.
\textbf{Watermark:} the defender watermarks all legitimate entries at write time via a unigram scheme~\citep{zhao2024watermark} ($z_{\text{thr}} = 4.0$, $\fpr \approx 3.2 \times 10^{-5}$), rejecting unwatermarked entries; this requires control of the ingestion pipeline.
\textbf{Validation:} rule-based pattern matching ($O(|c| \cdot |P|)$).
\textbf{Proactive:} computes mean cosine similarity against 16 domain probes, flagging entries exceeding $\tau = 0.19$.
\textbf{Composite:} weighted ensemble of the above: flag if $\sum_j w_j d_j(c) > \theta$.

\section{Discussion and limitations}
\label{sec:discussion}

\paragraph{No single defense dominates.}
Defense effectiveness depends on the attack's access model $\alpha$: watermarking detects unwatermarked entries but fails when auto-storage bypasses the watermarking step; \memsad{} excels against \minja{} but needs triggered calibration for \agentpoison{}; proactive detection is complementary.
The defender's uncertainty about $\alpha$ motivates the portfolio approach: composite defenses realize $\asrr^* = 0$ (Proposition~\ref{prop:equilibrium}).

\paragraph{Deployment and optimality.}
Lemma~\ref{lem:certified} provides per-entry certified detection; Theorem~\ref{thm:minimax_lower} establishes minimax optimality up to a $\log(1/\delta)$ factor.
The practical implication is to maximize the SNR $\rho$ via triggered calibration, which raises $\rho$ from $1.2$ to $3.8$ for \agentpoison{}.
We recommend a layered deployment: (i)~watermarking at ingestion, (ii)~rolling \memsad{} at retrieval, and (iii)~composite detection for high-value agents ($\sim$200\,ms total; Appendix~\ref{app:complexity}).

\paragraph{The synonym loophole.}
Proposition~\ref{prop:synonym} reveals a structural limitation of all embedding-based detectors: paraphrase-trained encoders map synonyms to nearly identical embeddings ($\epsilon_{\text{syn}} \approx 0$), creating a discrete evasion channel orthogonal to gradient coupling.
Closing this gap requires complementary features: lexical diversity, syntactic structure, or provenance metadata.

\paragraph{Broader implications.}
Memory poisoning is simultaneously easier than previously reported ($4\times$ under the intended protocol) and more defensible than expected (composite $\tpr = 1.00$, $\fpr = 0.00$), but no purely semantic defense is complete. The Stackelberg formulation provides a principled framework for this arms race.

\paragraph{Limitations.}
(i)~Our corpus contains 200 entries, validated up to 500 (Appendix~\ref{app:ablation}); production systems may contain $10^4$--$10^6$ entries, though \memsad{}'s $O(md)$ cost is independent of $|\cM|$.
(ii)~$\asra$ values are projected from original papers; production agents may behave differently.
(iii)~The regret bound (Theorem~\ref{thm:regret}) assumes slowly drifting queries; abrupt distribution shifts degrade calibration.
(iv)~We use a single primary encoder (MiniLM-L6); cross-encoder transfer is validated on 7 encoders (Appendix~\ref{app:encoder_gen}).

\section{Conclusion}
\vspace{-2pt}

We introduced \memsad{}, the first defense for memory poisoning in retrieval-augmented agents with formal guarantees: gradient coupling (Theorem~\ref{thm:gradient_coupling}) establishes that any monotone detector inherits the retrieval gradient, making continuous evasion impossible; the certified radius (Lemma~\ref{lem:certified}) gives checkable per-entry guarantees; and Le Cam's method (Theorem~\ref{thm:minimax_lower}) proves minimax optimality.
Equally important is what \memsad{} \emph{cannot} do: Proposition~\ref{prop:synonym} identifies exactly where continuous guarantees break down, showing that the gap between semantic similarity and lexical identity is the fundamental open frontier.
Closing this gap---perhaps via joint semantic-lexical detectors or provenance-aware memory architectures---is the key challenge for securing the next generation of persistent-memory agents.
Code and data will be released upon publication.

\section{Proofs of main results}
\label{app:proofs}

\subsection{Proof of Theorem~\ref{thm:gradient_coupling} (geometric interpretation)}
\label{app:proof_gradient}

Under L2-normalization, embeddings lie on $\mathbb{S}^{d-1}$. The Riemannian gradient of $f(e_c) = e_c^\top v$ on the sphere is $\nabla_{\mathbb{S}} f = v - (e_c^\top v) e_c = \Pi_{e_c}^{\perp} v$, the projection of $v$ onto the tangent space $T_{e_c}\mathbb{S}^{d-1}$. For any monotone transformation $g$, the chain rule gives $\nabla(g \circ f) = g'(f) \cdot \nabla f$, which has the same sign as $\nabla f$ since $g' > 0$. The converse (non-monotone $\cD$ admits evasion) follows because non-monotonicity creates a region where increasing retrieval similarity decreases detection score, giving the adversary a ``free'' direction in embedding space.

\subsection{Proof of Theorem~\ref{thm:minimax_lower} (detailed)}
\label{app:proof_minimax}

\paragraph{Setup.} Instantiate Le Cam's method with:
$H_0$: $s(c; \cH) \sim P_0 = \cN(\mu_0, \sigma^2)$ (benign);
$H_1$: $s(c; \cH) \sim P_1 = \cN(\mu_1, \sigma^2)$ (poisoned), $\mu_1 > \mu_0$.
Then $\KL(P_0 \| P_1) = (\mu_1 - \mu_0)^2 / (2\sigma^2) = \rho^2/2$.

\paragraph{Application.} For product distributions, $\KL(P_0^N \| P_1^N) = N\rho^2/2$. By Pinsker: $\TV(P_0^N, P_1^N) \leq \sqrt{N\rho^2/4} = \rho\sqrt{N}/2$. Le Cam gives minimax risk $R^* \geq \frac{1}{2}(1 - \TV) \geq \frac{1}{2}(1 - \rho\sqrt{N}/2)$. Total error $\geq 2R^* = 1 - \rho\sqrt{N}/2$.

\paragraph{Alternative via Fano.} For $M = 2$ hypotheses, Fano's inequality gives $P_e \geq 1 - (I(X; \Theta) + \log 2)/\log M = 1 - (N\rho^2/4 + \log 2)/\log 2$, where $I(X; \Theta) \leq \max_j \KL(P_j^N \| \bar{P}^N) \leq N\rho^2/4$ by the mixture bound. This yields a slightly weaker constant but extends to the $M$-ary setting (e.g., distinguishing $M$ attack types simultaneously).

\paragraph{Tightness.} For Gaussian $P_0, P_1$, the Neyman-Pearson optimal test achieves $\fpr + (1-\tpr) = 2\Phi(-\rho\sqrt{N}/2)$ where $\Phi$ is the Gaussian CDF. Comparing with the Le Cam bound: $2\Phi(-x) \approx \sqrt{2/\pi}\,e^{-x^2/2}/x$ for large $x$, confirming the $\Omega(1/\rho^2)$ sample complexity is tight for Gaussian scores.

\subsection{Proof of Theorem~\ref{thm:calibration_bound}}
\label{app:proof_calibration}

By Hoeffding on $[0,1]$: $|\hat{\mu}_N - \mu| \leq \sqrt{\log(2/\delta)/(2N)}$ w.p.\ $\geq 1 - \delta/2$. By chi-squared concentration: $|\hat{\sigma}_N - \sigma| \leq \sigma\sqrt{\log(2/\delta)/(2(N-1))}$ w.p.\ $\geq 1 - \delta/2$. Triangle inequality: $|\tau_N - \tau^*| \leq |\hat{\mu}_N - \mu| + \kappa|\hat{\sigma}_N - \sigma|$. Union bound yields the calibration bound in Theorem~\ref{thm:calibration_bound}.

\subsection{Proof of Theorem~\ref{thm:regret}}
\label{app:proof_regret}

Decompose: $|\hat{\tau}_t - \tau_t^*| \leq \underbrace{|\hat{\tau}_t - \tau_t^{\cH}|}_{\text{estimation}} + \underbrace{|\tau_t^{\cH} - \tau_t^*|}_{\text{drift}}$.
Estimation error: $O(\sigma(1+\kappa)/\sqrt{m_{\max}})$ by Theorem~\ref{thm:calibration_bound}.
Drift error: cosine similarity is 1-Lipschitz on $\mathbb{S}^{d-1}$, and the window mean averages over $m_{\max}$ steps with linearly accumulating drift, so $|\mu_t^{\cH} - \mu_t| \leq m_{\max}\Delta/2$ (worst case at the window boundary; the $1/2$ arises from averaging). This affects only the constant in $m_{\max}^*$ but not the rate.
Setting equal: $\sigma/\sqrt{m_{\max}} = m_{\max}\Delta$ gives $m_{\max}^* = (\sigma/\Delta)^{2/3}$ (up to constants), yielding per-step regret $O(\sigma^{2/3}\Delta^{1/3})$.

\subsection{Detection-evasion tradeoff surface}
\label{app:tradeoff_surface}

By Theorem~\ref{thm:gradient_coupling}, $\cD(e_c') < \cD(e_c)$ (evasion) implies $\cR(e_c') < \cR(e_c)$ (retrieval degradation) to first order. The tradeoff defines a surface in $(\epsilon, \theta, \Delta\cD)$ space where $\theta$ is the angle between perturbation $\delta$ and gradient $\nabla\cD$. Expanding to second order:
\begin{equation}
  \cD(e_c + \epsilon\delta) - \cD(e_c) = \epsilon\|\nabla\cD\|\cos\theta + \frac{\epsilon^2}{2}\delta^\top H_{\cD}\,\delta + O(\epsilon^3),
\end{equation}
where $H_{\cD}$ is the Hessian of the cosine similarity on the sphere. Under L2-normalization, $H_{\cD} = -(e_c^\top \enc(q^*))(I - e_c e_c^\top)/\|e_c\|^2$, giving eigenvalues $\lambda_i = -(e_c^\top \enc(q^*))$ for directions orthogonal to $e_c$ (and $0$ in the radial direction). The evasion region $\cD(e_c') < \cD(e_c)$ requires $\cos\theta < -\epsilon\|H_{\cD}\|/(2\|\nabla\cD\|)$, which contracts as $\epsilon \to 0$, confirming the infinitesimal impossibility of evasion without retrieval degradation.

\subsection{Proof of Theorem~\ref{thm:generalization} (detailed)}
\label{app:proof_generalization}

\paragraph{Setup.} Let $F_j$ and $F_{\text{new}}$ denote the CDFs of scores $s(c; \cH)$ under $P_j$ and $P_{\text{new}}$, respectively. The TPR at threshold $\tau$ is $\tpr(\tau) = 1 - F(\tau)$.

\paragraph{Wasserstein coupling.} By Kantorovich-Rubinstein duality:
\begin{equation}
  W_1(F_j, F_{\text{new}}) = \sup_{\|f\|_{\text{Lip}} \leq 1} |\EE_{F_j}[f] - \EE_{F_{\text{new}}}[f]| = \int_{-\infty}^{\infty} |F_j(s) - F_{\text{new}}(s)| \, ds.
\end{equation}
Since $\tpr_j = 1 - F_j(\tau)$ and $\tpr_{\text{new}} = 1 - F_{\text{new}}(\tau)$:
$|\tpr_j - \tpr_{\text{new}}| = |F_{\text{new}}(\tau) - F_j(\tau)| \leq W_1(F_j, F_{\text{new}})/\hat{\sigma}$,
where the $1/\hat{\sigma}$ normalization arises because the detection operates on the standardized score $(s - \hat{\mu})/\hat{\sigma}$ and the Wasserstein distance scales inversely with the standard deviation of the reference distribution.

\paragraph{Threshold estimation.} By Theorem~\ref{thm:calibration_bound}, $|\hat{\tau} - \tau^*| \leq \eta_N$ w.p.\ $\geq 1-\delta$, contributing an additional $O(1/\sqrt{N})$ term to the TPR bound via the score density at the threshold.

\paragraph{Non-asymptotic convergence.} The non-asymptotic bound follows from applying a uniform concentration argument (McDiarmid's inequality) over the rolling window, with each window contributing an independent $O(\sigma/\sqrt{m_{\max}})$ estimation error and accumulating $O(m_{\max}\Delta)$ drift.

\subsection{Stackelberg equilibrium existence}
\label{app:equilibrium}

\begin{proposition}[Equilibrium existence]
\label{prop:equilibrium}
The Stackelberg game $\cG$ (Definition~\ref{def:stackelberg}) admits a subgame-perfect equilibrium $(\pi_{\cD}^*, \cP^*)$ when $\Pi$ is restricted to threshold detectors with $\kappa \in [\kappa_{\min}, \kappa_{\max}]$.
\end{proposition}

\begin{proof}
The defender's strategy space $\Pi_\kappa = \{\pi_\kappa : \kappa \in [\kappa_{\min}, \kappa_{\max}]\}$ is compact. For fixed $\pi_\kappa$, the adversary's best-response set $\cB(\pi_\kappa) = \argmax_{\cP} \asrr(\cP \setminus \{p : s(p; \cH) > \hat{\mu} + \kappa\hat{\sigma}\}, \cQ_v)$ is nonempty (the empty set is always feasible) and upper hemicontinuous in $\kappa$ since $s(p; \cH)$ is continuous and the threshold $\hat{\mu} + \kappa\hat{\sigma}$ is linear in $\kappa$. The defender's objective $\max_{\cP \in \cB(\pi_\kappa)} \asrr(\cP, \cQ_v)$ is upper semicontinuous by the Berge maximum theorem. By the extreme value theorem, $\pi_{\cD}^* = \argmin_{\kappa \in [\kappa_{\min}, \kappa_{\max}]} \max_{\cP \in \cB(\pi_\kappa)} \asrr$ exists.
\end{proof}

In practice, the composite portfolio $(\pi_{\text{wm}} \lor \pi_{\text{SAD}} \lor \pi_{\text{pro}})$ achieves $\asrr^* = 0$ against all three attacks (Table~\ref{tab:defense_tpr_fpr}), representing the defender's ideal equilibrium where the adversary's best response yields zero utility.

\subsection{Finite-sample FPR concentration}
\label{app:fpr_concentration}

\begin{proposition}[FPR concentration]
\label{prop:fpr_concentration}
Let $c_1, \ldots, c_n$ be $n$ i.i.d.\ benign candidates with scores $s_i = s(c_i; \cH)$. Let $\widehat{\fpr}_n = n^{-1}\sum_{i=1}^n \mathbf{1}[s_i > \hat{\mu} + \kappa\hat{\sigma}]$. Under sub-Gaussian scores with parameter $\sigma^2$:
\begin{equation}
  \PP\!\left[|\widehat{\fpr}_n - \fpr| > \epsilon\right] \leq 2\exp\!\left(-\frac{n\epsilon^2}{2\fpr(1-\fpr)}\right) + 2\exp\!\left(-\frac{n\epsilon^2}{8}\right).
\end{equation}
The first term is the Bernstein bound on the empirical FPR; the second accounts for threshold estimation error (from Theorem~\ref{thm:calibration_bound}). For $\kappa = 2.0$ and $n = 1{,}000$: $\PP[|\widehat{\fpr} - 0| > 0.01] < 0.006$, consistent with the Clopper-Pearson CI $[0.000, 0.004]$ (Table~\ref{tab:fpr_validation}).
\end{proposition}

\section{Graph memory attacks}
\label{app:graph}

\begin{definition}[Graph memory system]
A \emph{graph memory} is $(\cG, \cM, \enc, k)$ where $\cG = (V, E, \phi)$ is a knowledge graph with attribute function $\phi : V \cup E \to \cV^*$. Retrieval combines embedding similarity with BFS traversal:
$\Ret_{\cG}(q, k, h) = \{v \in V : d_{\cG}(v, v^*) \leq h,\; v^* \in \argmax_{v'} \cosim(\enc(q), \enc(\phi(v')))\}$.
\end{definition}

Three structural attacks: \textbf{Hub insertion} (high-degree adversarial node), \textbf{edge hijack} (shortcuts to adversarial nodes), \textbf{subgraph cluster} (self-reinforcing adversarial community). Degree-anomaly detection flags $z_{\deg}(v) > \kappa_{\deg}$; adjacency contamination scoring detects edge-hijack via the fraction of recently added edges.

\section{Multi-agent propagation}
\label{app:propagation}

\begin{definition}[SIR propagation]
For $N$ agents sharing $\cM_{\text{shared}}$, the discrete-step SIR model~\citep{gu2024agentsmith} evolves: $S(t{+}1) = S(t) - \beta S(t) I(t)/N$, $I(t{+}1) = I(t) + \beta S(t) I(t)/N - \gamma I(t)$, $R(t{+}1) = R(t) + \gamma I(t)$, where $\beta$ is transmission rate and $\gamma$ recovery rate.
\end{definition}

Without defense, steady-state infection is $I_\infty/N = 1 - \gamma/\beta$ for $\beta > \gamma$. With \memsad{} quarantine ($\tpr = 1.00$ for \minja{}), $\gamma_{\text{eff}} = \gamma + \tpr \cdot \beta$; the condition $\beta > \gamma_{\text{eff}}$ is never satisfied, eliminating secondary infection.

\section{Distributional robustness analysis}
\label{app:dro}

We extend \memsad{}'s guarantees to the distributionally robust setting where the adversary can perturb the score distribution within a Wasserstein ball.

\begin{proposition}[Distributionally robust detection]
\label{prop:dro}
Let $\hat{P}_0$ be the empirical benign score distribution from $N$ calibration samples. For any adversarial score distribution $P_1$ and perturbation class $\cU = \{Q : W_1(Q, P_1) \leq \epsilon\}$:
\begin{equation}
  \inf_{Q \in \cU} \tpr(\tau; Q) \geq \tpr(\tau; P_1) - \frac{\epsilon}{\hat{\sigma}}.
\end{equation}
\end{proposition}

\begin{proof}
By Kantorovich-Rubinstein duality, for any 1-Lipschitz $f$: $|\EE_Q[f] - \EE_{P_1}[f]| \leq W_1(Q, P_1) \leq \epsilon$. The indicator $\mathbf{1}[s > \tau]$ has Lipschitz constant $1/\hat{\sigma}$ when evaluated on standardized scores $(s - \hat{\mu})/\hat{\sigma}$, yielding the bound.
\end{proof}

This guarantees that even if the adversary slightly perturbs their attack strategy (e.g., via randomized synonym selection), \memsad{}'s TPR degrades gracefully. For $\epsilon = 0.01\hat{\sigma}$: $\Delta\tpr \leq 0.01$, consistent with the observed stability in Table~\ref{tab:adaptive_sad}.

\begin{proposition}[PAC-style detection guarantee]
\label{prop:pac}
For $N$ i.i.d.\ calibration samples from $P_0$, with probability $\geq 1 - \delta$ over the calibration set, \memsad{}'s threshold $\hat{\tau}_N$ satisfies:
\begin{equation}
  \fpr(\hat{\tau}_N) \leq \fpr(\tau^*) + \sqrt{\frac{\log(2/\delta)}{2N}}, \qquad
  \tpr(\hat{\tau}_N) \geq \tpr(\tau^*) - \sqrt{\frac{\log(2/\delta)}{2N}},
\end{equation}
where $\tau^* = \mu + \kappa\sigma$ is the population-optimal threshold.
\end{proposition}

\begin{proof}
By the DKW inequality, $\sup_t |F_N(t) - F(t)| \leq \sqrt{\log(2/\delta)/(2N)}$ w.p.\ $\geq 1-\delta$, where $F_N, F$ are empirical and population CDFs. Since $\fpr(\tau) = 1 - F_0(\tau)$ and $\tpr(\tau) = 1 - F_1(\tau)$, the bound follows from the uniform convergence of the empirical CDF to the true CDF at the estimated threshold.
\end{proof}

At $N = 200$, $\delta = 0.05$: the excess error is $\leq 0.086$, meaning \memsad{}'s empirical FPR of $0.000$ implies true FPR $\leq 0.086$ with 95\% confidence. The Clopper-Pearson validation (Table~\ref{tab:fpr_validation}) provides a tighter bound of $0.004$ by exploiting the binomial structure.
