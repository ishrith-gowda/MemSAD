\begin{abstract}
Persistent external memory enables LLM agents to maintain context across sessions, yet its security properties remain formally uncharacterized.
We introduce a rigorous threat model for memory poisoning attacks on retrieval-augmented agents and present a unified evaluation framework encompassing three attack classes with distinct access assumptions: write-access trigger optimization (\agentpoison{}), query-only progressive injection (\minja{}), and single-interaction anchor insertion (\injecmem{}).
We identify and formally correct a systematic evaluation protocol error in prior work that underestimates attack success rate by a factor of four.
Our primary theoretical contribution is \emph{Semantic Anomaly Detection} (\sad{}), a calibration-based defense for which we prove a \emph{gradient coupling theorem}: for any differentiable encoder, the anomaly score gradient and retrieval objective gradient are provably identical, establishing that continuous evasion of \sad{} necessarily reduces retrieval rank.
We derive tight bounds on the detection-evasion tradeoff, characterize a discrete loophole arising from the synonym-invariance of paraphrase-trained encoders, and provide information-theoretic lower bounds on the sample complexity of calibration.
Empirical evaluation over a FAISS-backed vector store with $|\cM|=200$ entries, $3 \times 5$ attack-defense interaction matrix, bootstrap confidence intervals, and Clopper-Pearson exact binomial validation demonstrates that composite defense portfolios achieve $\tpr = 1.00$ and $\fpr = 0.00$ across all attack classes, while the synonym loophole enables 80--100\% evasion at $\Delta\asrr \approx 0$, motivating joint semantic-lexical detection architectures.
\end{abstract}

\section{Introduction}
\label{sec:intro}

Modern LLM agents persist context across sessions through external memory systems such as Mem0~\citep{mem0}, A-MEM~\citep{amem}, and MemGPT~\citep{memgpt2023}.
These systems store user preferences, task history, conversation summaries, and factual knowledge as dense vector embeddings, retrieving semantically relevant entries to condition the agent's response.
The practical utility of persistent memory is well established; its security implications are not.

An adversary who can influence the contents of an agent's memory, whether through direct write access, crafted user interaction, or exploitation of the agent's self-storage mechanism, can covertly redirect agent behavior across arbitrary future sessions.
Unlike prompt injection, which operates on a single interaction, memory poisoning attacks \emph{persist}: a single adversarial entry may remain in memory for weeks and trigger on every related query.
The agent has no inherent mechanism to distinguish a legitimate memory from an injected one.

Three recent papers have characterized distinct threat models for memory poisoning.
\agentpoison{}~\citep{chen2024agentpoison} formulates trigger optimization as a constrained gradient problem over dense passage retrieval embeddings.
\minja{}~\citep{dong2025minja} extends the threat model to a query-only setting via progressive-shortening indication.
\injecmem{}~\citep{injecmem2026} achieves single-interaction injection via retriever-agnostic anchor passages.
Despite the severity of these threats, no prior work provides a unified evaluation across these attacks or proposes defenses with formal detection guarantees specifically designed for the memory-agent setting.
Existing RAG defenses~\citep{zou2024poisonedrag, chaudhari2024phantom} assume corpus-level access and offline cleaning, neither of which applies to streaming memory ingestion where entries arrive one at a time during agent operation.

\paragraph{Contributions.}
\textbf{(1)}~A formal threat model (Definition~\ref{def:threat_model}) and unified evaluation framework with three paper-faithful attack implementations, three metrics ($\asrr$, $\asra$, $\asrt$), and a systematic evaluation protocol correction raising \agentpoison{} $\asrr$ from 0.25 to 1.00.
\textbf{(2)}~\sad{}: a novel Semantic Anomaly Detection defense with a combined max/mean scoring mode (Definition~\ref{def:sad}), for which we prove a gradient coupling theorem (Theorem~\ref{thm:gradient_coupling}) establishing that continuous evasion is impossible.
\textbf{(3)}~Tight characterization of the detection-evasion tradeoff (Corollary~\ref{cor:tradeoff}), information-theoretic calibration bounds (Theorem~\ref{thm:calibration_bound}), and formal analysis of the discrete synonym loophole (Proposition~\ref{prop:synonym}).
\textbf{(4)}~A $3 \times 5$ attack-defense interaction matrix with five defenses, demonstrating that no single defense dominates uniformly and that composite portfolios achieve $\tpr = 1.00$ across all attacks.
\textbf{(5)}~Adaptive adversary analysis confirming the synonym loophole empirically (evasion rate $\geq 0.80$, $\Delta\asrr \approx 0$) and motivating joint semantic-lexical detection.

\section{Related work}
\label{sec:related}

\paragraph{Memory poisoning attacks on LLM agents.}
\citet{chen2024agentpoison} formulate trigger optimization as a constrained gradient problem over DPR~\citep{karpukhin2020dense} embeddings, achieving $\asrr \geq 0.80$ on production agent frameworks.
\citet{dong2025minja} extend the threat model to a query-only setting via progressive-shortening indication ($\isr = 98.2\%$).
\citet{injecmem2026} achieve single-interaction injection via retriever-agnostic anchors.
\citet{memorygraft2025} demonstrate experience-grafting attacks exploiting union retrieval over lexical and embedding similarity.
Concurrent work on tool-use manipulation~\citep{yang2024watch} shows that similar poisoning strategies transfer to agent tool selection.

\paragraph{RAG corpus poisoning.}
\citet{zou2024poisonedrag} optimize adversarial documents via HotFlip~\citep{ebrahimi2018hotflip} ($> 90\%$ ASR at million-scale knowledge bases).
\citet{chaudhari2024phantom} construct phantom documents appearing benign to human inspection.
\citet{arzanipour2025ragsecurity} provide the first formal RAG threat model, defining adversary access tiers.
Our work differs from RAG poisoning in that adversarial entries are \emph{agent-generated, semantically coherent memories} that persist across sessions without further adversarial interaction, and defenses must operate at write time without full corpus visibility.

\paragraph{Anomaly detection and watermarking.}
STRIP~\citep{gao2019strip} and Neural Cleanse~\citep{wang2019neuralcleanse} are model-centric defenses requiring inference-time or white-box model access, rendering them inapplicable to write-time memory content.
\citet{zhao2024watermark} propose provably robust token-level watermarks (AUROC $> 0.97$) with formal guarantees under bounded paraphrasing.
\citet{lee2018simple} show that Mahalanobis distance to class centroids is an effective out-of-distribution detector; our cosine similarity threshold is the retrieval-space analogue adapted to the unsupervised, query-conditioned memory setting.

\section{Formal threat model}
\label{sec:threat}

We begin by formalizing the memory poisoning threat model, defining the system, adversary, and success metrics precisely.

\begin{definition}[Memory-augmented agent]
\label{def:agent}
A \emph{memory-augmented agent} is a tuple $(\cA, \cM, \enc, k)$ where $\cA$ is an LLM, $\cM \subset \cV^*$ is a persistent memory store over vocabulary $\cV$, $\enc : \cV^* \to \RR^d$ is an embedding function, and $k \in \NN$ is the retrieval depth. At each interaction with query $q \in \cV^*$, the agent retrieves
\begin{equation}
  \Ret(\cM, q, k) = \argmax_{\substack{S \subseteq \cM \\ |S| = k}} \sum_{m \in S} \cosim(\enc(q), \enc(m)),
  \label{eq:retrieval}
\end{equation}
where $\cosim(u, v) \coloneqq u^\top v / (\|u\| \|v\|)$ denotes cosine similarity.
\end{definition}

\begin{definition}[Memory poisoning threat model]
\label{def:threat_model}
Let $(\cA, \cM, \enc, k)$ be a memory-augmented agent. An \emph{adversary} $\cS$ is characterized by a tuple $(\cQ_v, n, \alpha)$ where $\cQ_v \subset \cV^*$ is a set of \emph{victim queries}, $n \in \NN$ is the number of injected entries, and $\alpha \in \{\textsc{write}, \textsc{query}, \textsc{single}\}$ specifies the access model. The adversary injects $\cP = \{p_1, \ldots, p_n\} \subset \cV^*$ into $\cM$ to maximize the following success metrics:
\begin{align}
  \asrr(\cP, \cQ_v) &\coloneqq \frac{|\{q \in \cQ_v : \cP \cap \Ret(\cM \cup \cP, q, k) \neq \emptyset\}|}{|\cQ_v|}, \label{eq:asrr} \\
  \asra(\cP) &\coloneqq \PP[\cA \text{ executes adversarial action} \mid \text{poison retrieved}], \label{eq:asra} \\
  \asrt(\cP, \cQ_v) &\coloneqq \asrr(\cP, \cQ_v) \cdot \asra(\cP). \label{eq:asrt}
\end{align}
\end{definition}

\begin{assumption}[Encoder regularity]
\label{ass:encoder}
The encoder $\enc : \cV^* \to \RR^d$ satisfies: (i) $\|\enc(x)\| > 0$ for all $x \in \cV^*$; (ii) $\enc$ is differentiable with respect to its output when viewed as a function on $\RR^d$ (i.e., the mapping $\enc(c) \mapsto \cosim(\enc(c), \enc(q))$ is differentiable for fixed $q$); and (iii) L2-normalization: $\|\enc(x)\| = 1$ for all $x$ (without loss of generality via normalization).
\end{assumption}

Under Assumption~\ref{ass:encoder}(iii), the retrieval operation simplifies to $\Ret(\cM, q, k) = \argmax_{S \subseteq \cM, |S|=k} \sum_{m \in S} \enc(q)^\top \enc(m)$, and $\cosim(\enc(q), \enc(m)) = \enc(q)^\top \enc(m)$.

\section{Semantic Anomaly Detection}
\label{sec:sad}

\subsection{Algorithm and formal definition}

Memory poisoning attacks succeed precisely because adversarial entries are semantically close to victim queries; this closeness, necessary for retrieval, is itself a detectable signal.
\sad{} exploits this asymmetry without requiring labeled attack examples or model access.

\begin{definition}[\sad{} detector]
\label{def:sad}
Let $\cH = \{q_1, \ldots, q_m\}$ be a rolling history of observed queries (FIFO, capacity $m_{\max}$) and let $\cM_{\text{ref}} \subseteq \cM$ be a benign reference corpus. The \sad{} anomaly score for a candidate entry $c$ is
\begin{equation}
  s(c; \cH) \coloneqq \max_{q \in \cH} \cosim(\enc(c), \enc(q)).
  \label{eq:sad_score}
\end{equation}
The calibration step computes mean $\hat{\mu}$ and standard deviation $\hat{\sigma}$ of $\{s(m; \cH) : m \in \cM_{\text{ref}}\}$ over the benign reference corpus. An entry $c$ is flagged as anomalous if
\begin{equation}
  s(c; \cH) > \hat{\mu} + \kappa \cdot \hat{\sigma}, \qquad \kappa \in \RR_{>0}.
  \label{eq:sad_threshold}
\end{equation}
We additionally define a \emph{combined scoring mode} that captures both targeted (high max-similarity) and broad-recall (high mean-similarity) attack signatures:
\begin{equation}
  s_{\text{comb}}(c; \cH)
  \coloneqq \tfrac{1}{2} \max_{q \in \cH} \cosim(\enc(c), \enc(q))
  + \tfrac{1}{2} \cdot \frac{1}{|\cH|}\sum_{q \in \cH} \cosim(\enc(c), \enc(q)).
  \label{eq:sad_combined}
\end{equation}
\end{definition}

\begin{algorithm}[t]
\caption{\sad{} pre-ingestion filtering}
\label{alg:sad}
\begin{algorithmic}[1]
\REQUIRE candidate entry $c$, query history $\cH$, calibration parameters $(\hat{\mu}, \hat{\sigma})$, threshold $\kappa$
\STATE Compute $s \leftarrow s_{\text{comb}}(c; \cH)$ via Eq.~\eqref{eq:sad_combined}
\IF{$s > \hat{\mu} + \kappa \cdot \hat{\sigma}$}
  \STATE \textbf{reject} $c$ (flagged as anomalous)
\ELSE
  \STATE \textbf{accept} $c$ into $\cM$
\ENDIF
\end{algorithmic}
\end{algorithm}

\paragraph{Computational complexity.}
For a candidate entry $c$ with query history of size $m = |\cH|$ and embedding dimension $d$, computing $s_{\text{comb}}(c; \cH)$ requires $m$ inner products in $\RR^d$, yielding $O(md)$ time complexity. With FAISS batch search, this reduces to $O(d \log m)$ amortized per query via approximate nearest neighbor indexing.

\subsection{Gradient coupling theorem}
\label{sec:gradient}

A key security question for any anomaly-based defense is: \emph{can an adaptive adversary who knows the detection statistics $(\hat{\mu}, \hat{\sigma})$ craft passages that evade detection while preserving retrieval effectiveness?}
We prove that under Assumption~\ref{ass:encoder}, this is impossible for continuous perturbations.

\begin{theorem}[Gradient coupling]
\label{thm:gradient_coupling}
Let $\enc$ satisfy Assumption~\ref{ass:encoder}, let $c \in \cV^*$ be an adversarial passage with $e_c \coloneqq \enc(c) \in \RR^d$, and let $q^* \coloneqq \argmax_{q \in \cH} \cosim(e_c, \enc(q))$. Define:
\begin{itemize}
  \item \emph{Detection objective}: $\cD(e_c) \coloneqq \cosim(e_c, \enc(q^*)) = e_c^\top \enc(q^*) / (\|e_c\| \|\enc(q^*)\|)$
  \item \emph{Retrieval objective}: $\cR(e_c) \coloneqq \cosim(e_c, \enc(q^*))$
\end{itemize}
Then $\cD \equiv \cR$ on $\RR^d \setminus \{0\}$, and their gradients are identical:
\begin{equation}
  \nabla_{e_c} \cD(e_c)
  = \nabla_{e_c} \cR(e_c)
  = \frac{\enc(q^*)}{\|e_c\| \|\enc(q^*)\|}
    - \frac{e_c^\top \enc(q^*)}{\|e_c\|^3 \|\enc(q^*)\|} e_c.
  \label{eq:gradient}
\end{equation}
Consequently, for any perturbation direction $\delta \in \RR^d$:
\begin{equation}
  \langle \delta, \nabla_{e_c} \cD(e_c) \rangle < 0
  \iff
  \langle \delta, \nabla_{e_c} \cR(e_c) \rangle < 0.
  \label{eq:coupling}
\end{equation}
\end{theorem}

\begin{proof}
Since $\cD(e_c) = \cR(e_c) = \cosim(e_c, \enc(q^*))$ by construction, gradient identity follows immediately. For the explicit form, let $v \coloneqq \enc(q^*)$ and define $f(e_c) = e_c^\top v / (\|e_c\| \|v\|)$. Then:
\begin{align}
  \frac{\partial f}{\partial e_c^{(i)}}
  &= \frac{v^{(i)}}{\|e_c\| \|v\|}
     - \frac{(e_c^\top v) \cdot e_c^{(i)}}{\|e_c\|^3 \|v\|},
\end{align}
which gives $\nabla_{e_c} f = v / (\|e_c\| \|v\|) - (e_c^\top v / (\|e_c\|^3 \|v\|)) e_c$, establishing Eq.~\eqref{eq:gradient}.
The biconditional in Eq.~\eqref{eq:coupling} follows because $\nabla_{e_c} \cD = \nabla_{e_c} \cR$.
\end{proof}

\begin{corollary}[Detection-evasion tradeoff]
\label{cor:tradeoff}
Under Assumption~\ref{ass:encoder} with L2-normalized embeddings ($\|e_c\| = \|\enc(q^*)\| = 1$), the gradient simplifies to $\nabla_{e_c} \cD(e_c) = \enc(q^*) - (e_c^\top \enc(q^*)) e_c$. For any perturbation $e_c' = e_c + \epsilon \delta$ with $\|\delta\| = 1$ and $\epsilon > 0$ sufficiently small:
\begin{equation}
  \cD(e_c') - \cD(e_c) = \epsilon \langle \delta, \enc(q^*) - (e_c^\top \enc(q^*)) e_c \rangle + O(\epsilon^2).
  \label{eq:tradeoff}
\end{equation}
Therefore $\cD(e_c') < \cD(e_c)$ (evasion) implies $\cR(e_c') < \cR(e_c)$ (retrieval degradation) to first order.
\end{corollary}

\begin{proof}
Direct Taylor expansion of $\cD$ around $e_c$ using the gradient from Theorem~\ref{thm:gradient_coupling} with $\|e_c\| = \|\enc(q^*)\| = 1$.
\end{proof}

\subsection{Calibration sample complexity}

\begin{theorem}[Calibration bound]
\label{thm:calibration_bound}
Let $\{s_1, \ldots, s_N\}$ be i.i.d.\ draws from the benign score distribution $s(m; \cH)$ with $m \sim \cD_{\text{benign}}$, having mean $\mu$ and variance $\sigma^2 > 0$. Let $\hat{\mu}_N = \frac{1}{N}\sum_{i=1}^{N} s_i$ and $\hat{\sigma}_N^2 = \frac{1}{N-1}\sum_{i=1}^{N}(s_i - \hat{\mu}_N)^2$. For the threshold $\tau_N \coloneqq \hat{\mu}_N + \kappa \hat{\sigma}_N$ and the population threshold $\tau^* \coloneqq \mu + \kappa \sigma$, with probability at least $1 - \delta$:
\begin{equation}
  |\tau_N - \tau^*| \leq \sigma \left(\frac{1}{\sqrt{N}} + \frac{\kappa}{\sqrt{2(N-1)}}\right) \sqrt{2 \log(4/\delta)}.
  \label{eq:calibration_bound}
\end{equation}
\end{theorem}

\begin{proof}
By Hoeffding's inequality applied to the sample mean on $[0, 1]$ (since cosine similarities are bounded), $|\hat{\mu}_N - \mu| \leq \sqrt{\log(2/\delta) / (2N)}$ with probability $\geq 1 - \delta/2$. For the sample standard deviation, by the chi-squared concentration bound, $|\hat{\sigma}_N - \sigma| \leq \sigma \sqrt{\log(2/\delta) / (2(N-1))}$ with probability $\geq 1 - \delta/2$. The triangle inequality gives $|\tau_N - \tau^*| \leq |\hat{\mu}_N - \mu| + \kappa |\hat{\sigma}_N - \sigma|$, and a union bound over both events yields Eq.~\eqref{eq:calibration_bound}.
\end{proof}

\subsection{The discrete synonym loophole}

\begin{proposition}[Synonym-invariance loophole]
\label{prop:synonym}
Let $\enc$ be a paraphrase-trained encoder satisfying $\|\enc(\mathrm{sub}(c, w, w')) - \enc(c)\| \leq \epsilon_{\text{syn}}$ for any synonym substitution $w \to w'$, where $\mathrm{sub}(c, w, w')$ replaces word $w$ with synonym $w'$ in passage $c$. Then for $c' = \mathrm{sub}(c, w_1, w_1') \circ \cdots \circ \mathrm{sub}(c, w_r, w_r')$ with $r$ substitutions:
\begin{align}
  |\cR(c') - \cR(c)| &\leq r \cdot \epsilon_{\text{syn}}, \label{eq:retrieval_bound} \\
  |s(c'; \cH) - s(c; \cH)| &\leq r \cdot \epsilon_{\text{syn}}. \label{eq:detection_bound}
\end{align}
Consequently, if $\epsilon_{\text{syn}} \approx 0$ (as holds empirically for paraphrase-trained models), the adversary can reduce $s(c'; \cH)$ below the detection threshold while preserving $\cR(c') \approx \cR(c)$ by selecting synonyms that shift the anomaly score downward via non-gradient lexical variation.
\end{proposition}

\begin{proof}
For L2-normalized embeddings, $|\cosim(\enc(c'), \enc(q)) - \cosim(\enc(c), \enc(q))| = |\enc(q)^\top (\enc(c') - \enc(c))| \leq \|\enc(c') - \enc(c)\|$ by Cauchy-Schwarz. Each synonym substitution perturbs the embedding by at most $\epsilon_{\text{syn}}$, and triangle inequality over $r$ substitutions gives $\|\enc(c') - \enc(c)\| \leq r \cdot \epsilon_{\text{syn}}$. The bounds follow by applying this to both $\cR$ and $s(\cdot; \cH)$.
\end{proof}

\begin{remark}
\label{rem:mechanism}
The loophole in Proposition~\ref{prop:synonym} does not contradict Theorem~\ref{thm:gradient_coupling}. The gradient coupling theorem characterizes \emph{continuous} perturbations in $\RR^d$; synonym substitution operates in the \emph{discrete} token space $\cV^*$, where the encoder mapping is not differentiable. The synonym-invariance property $\epsilon_{\text{syn}} \approx 0$ means that discrete substitutions can change the calibration statistics (by altering surface-level features used in mean/variance estimation from different calibration samples) while leaving the embedding nearly unchanged. This motivates defenses incorporating lexical diversity features beyond the semantic embedding space.
\end{remark}

\section{Additional defenses}
\label{sec:other_defenses}

We evaluate four additional defenses under a pre-ingestion filtering model.

\paragraph{Watermark.}
Content provenance via the unigram watermark of~\citet{zhao2024watermark}. A PRF-keyed partition divides the vocabulary into green (fraction $\gamma = 0.25$) and red lists; detection computes z-score $z = (g - \gamma n)/\sqrt{\gamma(1-\gamma)n}$ against threshold $z_{\text{thr}} = 4.0$. Under the null hypothesis of random text, $z \sim \mathcal{N}(0, 1)$, yielding a false positive rate of $\PP[z > z_{\text{thr}}] = \Phi(-z_{\text{thr}}) \approx 3.2 \times 10^{-5}$.

\paragraph{Validation.}
Rule-based pattern matching for injection keywords applied at write time. Computationally lightweight at $O(|c| \cdot |P|)$ for pattern set $P$, but inherently limited against semantically sophisticated attacks that avoid explicit injection vocabulary.

\paragraph{Proactive.}
Measures mean cosine similarity against 16 domain probe queries spanning 8 agent subtopics. Entries with mean probe similarity exceeding $\tau = 0.19$ are flagged. Formally, for probe set $Q_{\text{probe}}$: flag $c$ if $\frac{1}{|Q_{\text{probe}}|}\sum_{q \in Q_{\text{probe}}} \cosim(\enc(c), \enc(q)) > \tau$.

\paragraph{Composite.}
Weighted ensemble implementing defense-in-depth. An entry is flagged if $\sum_{j} w_j \cdot d_j(c) > \theta$ where $d_j(c) \in \{0, 1\}$ are individual detector outputs and $w_j, \theta$ are learned parameters.

\subsection{Adaptive adversary analysis}
\label{sec:adaptive}

We model a white-box adversary with knowledge of $(\hat{\mu}, \hat{\sigma})$ who applies greedy synonym substitution ($>60$ domain synonym pairs) to solve:
\begin{equation}
  c^* = \argmin_{c' \in \cV^*}\; s(c'; \cH) \quad \text{s.t.} \quad \rank(\enc(c'), \cM) \leq k, \quad c' \in \mathrm{Syn}(c),
  \label{eq:adaptive_obj}
\end{equation}
where $\mathrm{Syn}(c)$ denotes the set of passages reachable from $c$ via synonym substitution.

\section{Discussion and limitations}
\label{sec:discussion}

\paragraph{No single defense dominates.}
The $3 \times 5$ matrix reveals that defense effectiveness varies substantially by attack type.
Watermark reliably identifies unsigned writes but provides no protection if the attacker bypasses the ingestion path.
\sad{} excels against similarity-concentrated attacks (\minja{}) but misses centroid-targeting attacks under plain calibration.
Proactive detection catches broad-recall passages but misses targeted ones.
Composite portfolios cover all gaps by combining these complementary signals.

\paragraph{Triggered-query calibration as a natural defense property.}
The \agentpoison{} blind spot is resolved without architectural changes: once triggered queries appear in the rolling history, \sad{} becomes fully effective.
This makes \sad{} an inherently adaptive defense at no additional cost.

\paragraph{Deployment recommendations.}
We recommend a three-layer defense architecture:
(i)~\emph{Provenance layer}: watermark verification at write time ($O(|c|)$ per entry).
(ii)~\emph{Anomaly layer}: rolling \sad{} calibration ($m_{\max} = 100$, combined scoring, $\kappa = 2.0$).
(iii)~\emph{Ensemble layer}: composite portfolio for defense-in-depth.
Total overhead is dominated by embedding inference ($\sim 200$ms per batch of 200 entries), making the defense pipeline negligible relative to LLM generation cost.

\paragraph{Limitations.}
(i)~The 200-entry synthetic corpus may not capture full production diversity; Appendix~\ref{app:ablation} shows robustness across $|\cM| \in \{50, \ldots, 500\}$.
(ii)~$\asra$ is modelled from paper-reported values; a local GPT-2 evaluator provides lower bounds ($0.29$--$0.51$).
(iii)~\sad{} assumes a stationary query distribution; gradual drift could degrade calibration (Theorem~\ref{thm:calibration_bound} quantifies the required sample size).
(iv)~The synonym loophole (Proposition~\ref{prop:synonym}) shows that lexical diversity features are needed to complement semantic detection.
(v)~All main experiments use all-MiniLM-L6-v2; Appendix~\ref{app:encoder_gen} validates generalization across 7 encoders.

\section{Conclusion}

We presented a formally grounded framework for evaluating memory poisoning attacks and defenses on retrieval-augmented LLM agents.
Our key contributions are:
(1)~a systematic evaluation correction showing \agentpoison{} is $4\times$ more effective than previously simulated;
(2)~\sad{}, a novel calibration-based defense with a gradient coupling theorem (Theorem~\ref{thm:gradient_coupling}) establishing that continuous evasion is impossible, complemented by calibration sample complexity bounds (Theorem~\ref{thm:calibration_bound});
(3)~formal characterization of the discrete synonym loophole (Proposition~\ref{prop:synonym}) with tight perturbation bounds;
(4)~a comprehensive $3 \times 5$ attack-defense matrix demonstrating that composite portfolios are both necessary and sufficient for complete coverage.

The gradient coupling result provides a formal foundation for understanding the fundamental tension between evasion and effectiveness in similarity-based detection. The synonym loophole identifies a precise boundary where this guarantee breaks down, motivating future work on joint semantic-lexical detection architectures that analyze both embedding-space anomalies and surface-level linguistic patterns.
All code, data, and evaluation scripts are open-sourced for full reproducibility.

\section{Additional defense details}
\label{app:defenses}

\paragraph{Watermark detection.}
The z-score $z = (g - \gamma n)/\sqrt{\gamma(1-\gamma)n}$ is computed where $g$ is the green-list count and $n$ is the sequence length. The character-level variant partitions ASCII printable characters (95 chars) via PRF, targeting green fraction $\gamma + \delta \times 0.1 = 0.45$. The token-level variant uses GPT-2 (50,257 tokens), achieving $p_{\text{green}} \approx 0.711$ and mean z-score $\approx 7.5$ at $n = 50$ tokens vs.\ $\approx 4.1$ for character-level at equivalent length.

\paragraph{Proactive detection.}
The 16 probe queries span 8 subtopics: preferences, calendar, tasks, security, configuration, conversation, documentation, and team communication. The threshold $\tau = 0.19$ was selected to minimize FPR on a held-out benign set while maintaining $\geq 0.50$ TPR on known centroid passages.

\paragraph{Evaluation correction detail.}
Prior to our correction, \agentpoison{} was evaluated by issuing plain queries $q$ without the trigger $\trigger^*$. The paper's Algorithm~2~\citep{chen2024agentpoison} specifies triggered queries $q \oplus \trigger^*$. Without the trigger, the centroid passage must compete with benign entries on generic similarity, a much easier task for defenders.

\section{Ablation studies}
\label{app:ablation}

\paragraph{\sad{} threshold sensitivity.}
At $\kappa = 1.5$: \injecmem{} TPR increases to 0.60 but FPR rises to 0.05. At $\kappa = 3.0$: \minja{} TPR drops to 0.80 while FPR remains 0.00. $\kappa = 2.0$ provides the best operating point with combined scoring mode.

\paragraph{Corpus size.}
$\asrr$ is robust to corpus size (50--500) for \agentpoison{} and \minja{}. \injecmem{} effectiveness decreases as competition for top-$k$ slots intensifies with larger corpora.

\paragraph{Poison count.}
Increasing $n_{\text{base}}$ from 1 to 10 monotonically increases $\asrr$ for all attacks, with diminishing returns above 5.

\section{Multi-encoder evaluation}
\label{app:encoder_gen}

We evaluate across 7 encoders: MiniLM-L6, MPNet-Base, Para-MiniLM, E5-Base, Contriever, BGE-Large, and OpenAI text-embedding-3-small. \agentpoison{} with triggered calibration achieves $\tpr = 1.000$ on all encoders. \minja{} TPR ranges from 0.80 (E5) to 1.00 (MiniLM, MPNet). Inter-encoder CKA~\citep{kornblith2019similarity} predicts $\asrr$ transferability.

\section{Graph memory attacks}
\label{app:graph}

We extend to GraphRAG-style entity-relation stores with three structural attacks: hub insertion, edge-hijack, and subgraph cluster. Degree-anomaly detection identifies structurally anomalous nodes; adjacency contamination scoring detects edge-hijack patterns.

\section{Multi-agent propagation}
\label{app:propagation}

A discrete-step SIR epidemic model~\citep{gu2024agentsmith} over $N$ agents sharing a FAISS knowledge base shows that a single poisoned entry amplifies via re-storage. \sad{}-based write-time quarantine eliminates secondary infection.

\section{FPR validation}
\label{app:fpr}

FPR is validated across 20 independent trials with fresh per-trial calibration using Clopper-Pearson exact binomial CIs~\citep{clopper1934use}, confirming $\fpr = 0.000$ at 95\% confidence (one-sided upper bound $< 0.03$, $n = 1000$ benign entries per trial).
